# To Salvo
Este Repositório publico do site [https://tosalvo.org](https://tosalvo.org), uma iniciativa para ajudar as vitimas da tragédia da inundação do Rio Grande do Sul de maio de 2024.

## Para rodar


```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
